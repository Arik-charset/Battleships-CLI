A simple Battleships game played in the CLI. Made using C#

Local multiplayer. Take turns placing and attacking ships.

This was built over a few days, a few hours at a time. 

The code has some potential for improvements. For example, the use of currentBoard and opponentBoard feels a bit clunky, especially the switching. This could be made into a method instead for better readability and a lover chance for logic mistakes.

Possible features that could be implement include: Arrow key selection for coordinates, multiplayer over the local network instead of just the local machine.

The current release is only for win-x64. You are always free to build other runtimes, just clone the repo into Visual Studio and press "Build".
